## Prompt Comparison (Phase 3)

This section compares outputs generated by different prompt strategies to evaluate improvements in reasoning, structure, and IAM decision accuracy.

### Test Inputs

1. QA needs write access to production DB  
2. Need read access to test database  
3. User logged in from India and US within 1 hour  
4. I urgently need elevated access to production db for hotfix  

### Comparison Table

| Input | Prompt 1 Output | Prompt 2 Output | Prompt 3 Output | Observation |
|------|----------------|----------------|----------------|------------|
| QA write prod | Deny (text) | Classification only | Deny + risk | Prompt 3 adds structure + risk |
| Read test DB | Uncertain | Classification only | Approve | Prompt 3 gives clear decision |
| India-US login | Deny  | Anomaly detected | Escalate  | Prompt 3 handles anomaly correctly |
| Hotfix access | Deny / vague | Classification | Escalate  | Prompt 3 handles high-risk scenario |

### Improvements Observed

- Prompt 1 produced unstructured and inconsistent outputs  
- Prompt 2 improved classification but lacked decision capability  
- Prompt 3 provided structured outputs, better reasoning, and correct IAM behavior  

### New Failure Modes

- Possible hallucination without real policy grounding  
- Output format inconsistencies  
- Limited accuracy for ambiguous inputs  

### Final Prompt Strategy

Prompt 3 is selected as the default because it provides structured, consistent, and actionable outputs aligned with IAM workflows.
